A fun global dating, marriage, and family simulation system. **27 commands.**

## Profile & Matching

| Command | Description |
|---|---|
| `dating profile create` | Create your dating profile |
| `dating profile view` | View your (or another user's) dating profile |
| `dating match` | Find a potential match |
| `dating likes` | View who liked your profile |
| `dating matches` | View your current matches |

## Marriage & Family

| Command | Description |
|---|---|
| `marry` | Propose marriage to a user |
| `divorce` | Divorce your partner |
| `divorceall` | Divorce all partners (if applicable) |
| `propose` | Send a marriage proposal |
| `couplename` | Generate a couple name |
| `anniversary` | View your anniversary date |
| `vow` | Set a marriage vow |
| `adopt` | Adopt another user as family |
| `disown` | Remove an adopted family member |
| `children` | View your adopted children |
| `family` | View your full family tree |
| `gift` | Send a gift to your partner |
| `gift catalog` | View available gifts |

## Fun Extras

| Command | Description |
|---|---|
