Casual fun commands for your server. **~15 commands.**

## Commands

| Command | Description |
|---|---|
| `8ball` | Ask the magic 8-ball a question |
| `dice` | Roll a dice |
| `truth` | Get a random Truth prompt |
| `dare` | Get a random Dare prompt |
| `iq` | Get a random (joke) IQ score |
| `ship` | "Ship" two users together with a compatibility % |
| `image` | Fetch a random themed image |
| `gif` | Fetch a random themed GIF |
| `weather` | Check the weather for a location |
| `translate` | Translate text between languages |
| `iplookup` | Look up information about an IP address |
| `fakeban` | Prank-ban a user (fake, cosmetic only) |
| `cute` | Get a random cute image |
| `mydog` | Get a random dog image |

<Note>
A few additional joke/rating-style commands exist in this module that aren't listed here as they use casual/informal language not suited for public docs. Run `,help fun` in Discord for the full live list.
</Note>
