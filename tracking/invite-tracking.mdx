---
title: Invite Tracking
description: 'Tracks valid server invites.'
---

Track who invited whom, and reward top inviters. **13 commands.**

## Commands

| Command | Description |
|---|---|
| `invitetrack` | View invite tracking status |
| `invites` | View a user's invite count |
| `inviter` | Check who invited a specific member |
| `invitetop` | View the invite leaderboard |
| `inviteinfo` | View detailed invite information |
| `sync` | Sync/re-scan server invites |
| `addinvites` | Manually add invites to a user |
| `removeinvites` | Manually remove invites from a user |
| `clearinvites` | Clear a user's invite count |
| `setinviterole` | Set a role reward for invite milestones |
| `unsetinviterole` | Remove an invite role reward |
| `viewinviteroles` | View all configured invite role rewards |
| `setaltthreshold` | Set the account-age threshold to flag alts |
