title: Message Tracking
description: 'Tracks total messages and message leaderboard.'
---

Track member message activity and view leaderboards. **8 commands.**

## Commands

| Command | Description |
|---|---|
| `msgtrack` | View message tracking status |
| `msgstats` | View a user's message statistics |
| `leaderboard` | View the message leaderboard |
| `categories` | View message stats by category |
| `livelb deploy` | Deploy a live-updating leaderboard |
| `msgreset` | Reset message tracking stats |
| `addmessages` | Manually add messages to a user's count |
| `removemessages` | Manually remove messages from a user's count |
