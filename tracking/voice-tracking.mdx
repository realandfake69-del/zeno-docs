Track voice channel activity and view leaderboards. **8 commands.**

## Commands

| Command | Description |
|---|---|
| `voicetrack` | View voice tracking status |
| `vcstats` | View a user's voice channel statistics |
| `voiceleaderboard` | View the voice activity leaderboard |
| `vcategories` | View voice stats by category |
| `voicelivelb deploy` | Deploy a live-updating voice leaderboard |
| `voicereset` | Reset voice tracking stats |
| `addvctime` | Manually add VC time to a user |
| `reducevctime` | Manually reduce a user's VC time |
