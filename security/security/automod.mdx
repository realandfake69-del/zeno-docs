Automatically filter spam, bad links, swearing, and more. **14 commands.**

## Commands

| Command | Description |
|---|---|
| `automod wizard` | Run the guided AutoMod setup wizard |
| `automod rules` | View current AutoMod rules |
| `automod whitelist` | Whitelist users/roles/channels from AutoMod |
| `automod antispam` | Configure anti-spam detection |
| `automod nsfw` | Configure NSFW content filtering |
| `automod caps` | Configure excessive caps filtering |
| `automod invite` | Block Discord invite links |
| `automod link` | Block or filter general links |
| `automod swear` | Configure swear word filtering |
| `automod mention` | Configure mass mention protection |
| `automod reset` | Reset all AutoMod settings |
| `automod logging` | Set the AutoMod log channel |
| `automod setup` | Initial AutoMod setup |
| `automod config` | View/edit full AutoMod configuration |
