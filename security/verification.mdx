Gate new members behind a verification flow. **12 commands.**

## Commands

| Command | Description |
|---|---|
| `verify setup` | Start the verification setup wizard |
| `verify panel` | Send the verification panel/button |
| `verify config` | View or edit verification settings |
| `verify disable` | Disable the verification system |
| `verify altaction` | Set action to take on detected alt accounts |
| `verify stats` | View verification statistics |
| `verify guide` | Show the verification setup guide |
| `verify bypass` | Bypass verification for a user/role |
| `verify blacklist add` | Add a user to the verification blacklist |
| `verify blacklist remove` | Remove a user from the blacklist |
| `verify blacklist list` | View all blacklisted users |
| `verify reset` | Reset verification configuration |
