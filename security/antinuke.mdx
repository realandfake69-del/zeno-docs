Protect your server from nukes, raids, and malicious admin actions. **18 commands.**

## Commands

| Command | Description |
|---|---|
| `antinuke setup` | Run the initial Antinuke setup wizard |
| `antinuke config` | View or edit current Antinuke configuration |
| `antinuke features` | Enable/disable specific protection modules |
| `antinuke manage` | Manage trusted admins for Antinuke |
| `antinuke punishment` | Set punishment type (ban/kick/timeout) |
| `antinuke limits` | Configure action limits before triggering |
| `whitelist user` | Whitelist a user from Antinuke actions |
| `whitelist role` | Whitelist a role from Antinuke actions |
| `antinuke unquarantine` | Remove a user from quarantine |
| `antinuke antibetray` | Protect against owner/admin betrayal |
| `antinuke zplus` | Enable enhanced protection tier |
| `antinuke barrier` | Set up an extra protection barrier |
| `wl violations` | View whitelist violation logs |
| `wl violations info` | Get details on a specific violation |
| `panicmode enable` | Lock down the server instantly |
| `panicmode disable` | Disable panic mode |
| `mainrole` | Set the main/owner role |
| `extraowner` | Add an extra owner for Antinuke purposes |

<Info>
  Run `,antinuke setup` first before configuring any other Antinuke settings.
</Info>
