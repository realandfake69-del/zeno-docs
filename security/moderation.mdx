title: Moderation
description: 'Manage users and channel restrictions.'
---

Core moderation toolkit — bans, mutes, warns, locks, and snipe utilities. **40 commands.**

## User Actions

| Command | Description |
|---|---|
| `ban` | Ban a user |
| `tempban` | Temporarily ban a user |
| `kick` | Kick a user |
| `mute` | Mute a user |
| `unmute` | Unmute a user |
| `unmuteall` | Unmute all muted users |
| `unban` | Unban a user |
| `unbanall` | Unban all banned users |
| `nick` | Change a member's nickname |
| `jail` / `setup jail` | Jail a user / configure the jail system |
| `unjail` | Remove a user from jail |
| `timeout` | Timeout a member |
| `untimeout` | Remove a timeout |

## Warnings

| Command | Description |
|---|---|
| `warns` | View a user's warnings |
| `warnconfig` | Configure the warning system |
| `warnrole` | Set a role applied on warning |
| `clearwarns` | Clear a user's warnings |

## Channel Controls

| Command | Description |
|---|---|
| `clear` | Bulk delete messages |
| `nuke` | Recreate a channel (clears all messages) |
| `lock` / `unlock` | Lock/unlock a channel |
| `lockall` / `unlockall` | Lock/unlock all channels |
| `hide` / `unhide` | Hide/unhide a channel |
| `hideall` / `unhideall` | Hide/unhide all channels |
| `slowmode` | Set channel slowmode |
| `setprefix` | Change the server's command prefix |

## Panic & Snipe

| Command | Description |
|---|---|
| `panicmode` / `panicmode off` | Lock down / restore the server |
| `snipe` | View the last deleted message |
| `editsnipe` | View the last edited message |
| `reactionsnipe` | View the last removed reaction |

## Emoji Tools

| Command | Description |
|---|---|
| `steal` | Steal an emoji into your server |
| `addemoji` | Add a single emoji |
| `addmultiple` | Add multiple emojis at once |
| `enlarge` | Enlarge an emoji to view it in full size |
| `emotes` | List all server emojis |
