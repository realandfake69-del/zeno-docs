Customize your Zeno profile card, bio, and badges. **8 commands.**

## Commands

| Command | Description |
|---|---|
| `profile` | View your profile card |
| `bio set` | Set your profile bio |
| `bio clear` | Clear your profile bio |
| `profile setbg` | Set a custom profile background |
| `setaccent` | Set your profile accent color |
| `badge list` | View available badges |
| `badge add` | Add a badge to a user (admin) |
| `badge remove` | Remove a badge from a user (admin) |
