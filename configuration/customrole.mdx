---
title: CustomRole
description: 'User-assignable custom role system.'
---

Let members create/claim their own custom roles, plus preset role shortcuts. **10 commands.**

## Setup

| Command | Description |
|---|---|
| `setup config` | View custom role configuration |
| `setup reset` | Reset custom role configuration |
| `setup reqrole` | Set the role required to use custom roles |
| `setup create` | Create a new custom role |
| `setup delete` | Delete a custom role |

## Preset Roles

| Command | Description |
|---|---|
| `staff` | Assign/toggle the Staff preset role |
| `girl` | Assign/toggle the Girl preset role |
| `friend` | Assign/toggle the Friend preset role |
| `vip` | Assign/toggle the VIP preset role |
| `guest` | Assign/toggle the Guest preset role |
