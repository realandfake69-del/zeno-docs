---
title: Media
description: 'Image-only channel filtering.'
---

Restrict channels to media/image-only content, with bypass controls. **12 commands.**

## Commands

| Command | Description |
|---|---|
| `media` | View media filter status |
| `mediaa` | Alternate media command |
| `media add` | Add a channel to media-only mode |
| `media remove` | Remove a channel from media-only mode |
| `media list` | List all media-only channels |
| `media bypass` | View bypass settings |
| `media bypass user add` | Allow a user to bypass the filter |
| `media bypass role add` | Allow a role to bypass the filter |
| `media bypass user remove` | Remove a user's bypass |
| `media bypass role remove` | Remove a role's bypass |
| `media bypass user list` | List users with bypass |
| `media bypass role list` | List roles with bypass |
