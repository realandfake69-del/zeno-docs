Set up a full support ticket system for your server. **15 commands.**

## Commands

| Command | Description |
|---|---|
| `ticket setup` | Run the ticket system setup wizard |
| `ticket edit` | Edit ticket system settings |
| `ticket delete` | Delete the ticket configuration |
| `ticket sendpanel` | Send the ticket creation panel |
| `ticket paneldelete` | Delete a ticket panel |
| `ticket list` | List all open tickets |
| `ticket close` | Close a ticket |
| `ticket remove` | Remove/delete a ticket channel |
| `ticket transcript` | Generate a transcript of a ticket |
| `ticket reopen` | Reopen a closed ticket |
| `ticket rename` | Rename a ticket channel |
| `ticket adduser` | Add a user to a ticket |
| `ticket removeuser` | Remove a user from a ticket |
| `ticket dm` | DM the ticket creator |
| `ticket blacklist` | Blacklist a user from creating tickets |
