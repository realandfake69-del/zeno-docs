---
title: VanityRole
description: 'Roles for custom status vanity.'
---

Automatically assign a role to members who have your server's vanity URL in their status. **3 commands.**

## Commands

| Command | Description |
|---|---|
| `vanityroles setup` | Set up the vanity role system |
| `vanityroles toggle` | Enable/disable the vanity role system |
| `vanityroles config` | View or edit vanity role configuration |
