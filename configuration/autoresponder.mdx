title: Autoresponder
description: 'Create custom auto-replies.'
---

Set up automatic replies triggered by keywords or phrases. **8 commands.**

## Commands

| Command | Description |
|---|---|
| `autoresponder list` | List all configured autoresponders |
| `autoresponder add` | Add a new trigger/response pair |
| `autoresponder edit` | Edit an existing autoresponder |
| `autoresponder remove` | Remove an autoresponder |
| `autoresponder info` | View details of a specific autoresponder |
| `autoresponder enable` | Enable the autoresponder system |
| `autoresponder disable` | Disable the autoresponder system |
| `autoresponder` | Base command / shows help |
