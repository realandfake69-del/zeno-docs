title: Tagrole
description: 'Roles based on clan tags.'
---

Automatically assign roles based on a member's clan tag. **5 commands.**

## Commands

| Command | Description |
|---|---|
| `tagrole setup` | Set up the tagrole system |
| `tagrole toggle` | Enable/disable tagrole |
| `tagrole config` | View or edit tagrole configuration |
| `tagrole mode` | Set matching mode for tags |
| `tagrole reset` | Reset tagrole configuration |
