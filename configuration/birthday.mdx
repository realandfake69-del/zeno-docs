Track member birthdays and get automatic announcements. **8 commands.**

## Commands

| Command | Description |
|---|---|
| `birthday set` | Set your birthday |
| `birthday show` | Show a member's birthday |
| `birthday next` | Show the next upcoming birthday |
| `birthday calendar` | View the full birthday calendar |
| `birthday setup` | Configure the birthday announcement channel |
| `birthday list` | List all registered birthdays |
| `birthday delete` | Remove your birthday |
| `birthday month` | View all birthdays in a given month |
