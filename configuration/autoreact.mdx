title: Autoreact
description: 'Setup auto-reactions for keywords, and autorole for new members.'
---

Automatically react to keywords, and auto-assign roles to new members. **10 commands.**

## Autoreact

| Command | Description |
|---|---|
| `autoreact add` | Add a keyword trigger with a reaction |
| `autoreact edit` | Edit an existing autoreact |
| `autoreact remove` | Remove an autoreact |
| `autoreact enable` | Enable the autoreact system |
| `autoreact disable` | Disable the autoreact system |
| `autoreact` | Base command / shows help |

## Autorole

| Command | Description |
|---|---|
| `autorole config` | Configure autorole settings |
| `autorole humans` | Set the autorole applied to human members |
| `autorole bots` | Set the autorole applied to bot accounts |
| `autorole` | Base command / shows help |
