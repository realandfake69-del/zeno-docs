Unlock exclusive customization features with Premium. **11 commands.**

## Commands

| Command | Description |
|---|---|
| `premium activate` | Activate a premium key for the server |
| `premium stats` | View premium status and stats |
| `premium setnick` | Set the bot's nickname (premium) |
| `premium setav` | Set the bot's avatar (premium) |
| `premium setbanner` | Set the bot's banner (premium) |
| `premium setbio` | Set the bot's bio (premium) |
| `premium setseparator` | Customize the help menu separator style |
| `premium np` | View "now playing" style status settings |
| `premium np add` | Add a now-playing style entry |
| `premium np remove` | Remove a now-playing style entry |
| `premium np list` | List all now-playing style entries |
