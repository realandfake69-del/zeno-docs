---
title: Autopurge
description: 'Automatic channel message cleaner.'
---

Automatically clear messages in a channel on a schedule. **3 commands.**

## Commands

| Command | Description |
|---|---|
| `autopurge setup` | Set up autopurge for a channel |
| `autopurge remove` | Remove autopurge from a channel |
| `autopurge config` | View or edit autopurge settings |
